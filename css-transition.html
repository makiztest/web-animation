CSS Transition

is a module of CSS that lets you create gradual transitions between the values of specific CSS properties. The behavior of these transitions can be controlled by specifying their timing function, duration, and other attributes.

Let's Start

Transition is a shorthand property for transition-property, transition-duration, transition-timing-function, and transition-delay.

  • transition-property the property you want to transition.
  • transition-durationspecifies the number of seconds or milliseconds a transition animation should take to complete. By default, the value is 0s, meaning that no animation will occur. always use ms(milliseconds) 2s or 2000ms
  • transition-timing-function is used to describe how the intermediate values of the CSS properties being affected by a transition effect are calculated. This in essence lets you establish an acceleration curve, so that the speed of the transition can vary over its duration.
  • transition-delay specifies the amount of time to wait between a change being requested to a property that is to be transitioned and the start of the transition effect.
CSS

  transition: property duration timing-function delay

  /* long form */
  transition-property: color;
  transition-duration: 2s;

  /* short form*/
  transition: color 2s 100ms;
transition: margin-right 3000ms;
transition: margin-right 3000ms;
